FreeCodeCamp API Basejump: Image Search Abstraction Layer

User stories:

  • I can get the image URLs, alt text and page urls for a set of images relating to a given search string.
  • I can paginate through the responses by adding a ?offset=2 parameter to the URL.
  • I can get a list of the most recently submitted search strings.

Example creation usage:

http://urlshortener-ms.herokuapp.com/new/https://www.google.com
http://urlshortener-ms.herokuapp.com/new/http://foo.com:80

Example creation output

{ "original_url":"http://foo.com:80", "short_url":"http://urlshortener-ms.herokuapp.com/3" }

Usage:

http://urlshortener-ms.herokuapp.com/1

Will redirect to:

https://www.google.com/
